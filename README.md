# FU Web

Frontend da aplicação de catálogo/enciclopédia para Fabula Ultima, construído com React, TypeScript, Vite e Cloudflare.

## Stack

- React
- TypeScript
- Vite
- React Router
- Vitest
- Cloudflare Workers / Wrangler

## Requisitos

- Node.js
- npm

## Instalação

````bash
npm install

## Formatação

Formatar o projeto:

```bash
npm run format
````
